# Informe_Educacion

Proyecto Overleaf liviano para el informe `Remuneracion Laboral por Nivel Educativo en Colombia 2008--2025`.

Fuente reproducible del analisis: `CJC-Monitor/Code/02.9_InformeRemuneracionEducacionGEIH.py`.

Este paquete incluye solo los insumos necesarios para compilar en Overleaf: `main.tex`, secciones LaTeX, figuras PNG y tablas CSV de respaldo. No incluye microdatos GEIH.
